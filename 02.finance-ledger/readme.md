# Personal Finance Ledger
 
**The question:** Where does my money actually go each month — and does tracking it category by category reveal anything surprising?
 
**What I did:** Built a file-backed CLI expense tracker that logs transactions by category, date, and amount. The ledger persists between sessions using a JSON file, supports adding/deleting entries, and generates a monthly summary broken down by category showing total spend, percentage of monthly budget, and biggest single expense per category.
